# Coding Spotify 🎵

A simple *Spotify Clone* built using *HTML, CSS, and JavaScript*.  
This project replicates the basic layout and functionality of Spotify’s music player interface.

---

## 🚀 Features
- Responsive design for desktop and mobile
- Play, pause, and skip controls
- Progress bar with real-time updates
- Playlist section with sample tracks
- Clean and modern UI inspired by Spotify

---

## 🛠️ Technologies Used
- *HTML5* – structure and content
- *CSS3* – styling and layout
- *JavaScript (ES6)* – interactivity and functionality
